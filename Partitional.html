Partitional algorithms assign a centroid to each group and then iteratively allocate the closest points to that centroid to form a cluster.
A Gaussian mixture model is a soft clustering algorithm that assigns various probabilities to points, making the algorithm more flexible. Clusters are formed due to different Gaussian distributions. Partitional algorithms and the Gaussian mixture model require the user to specify 'k', the input amount of clusters.

The goal of clustering is to find a structure in a collection of unlabelled data. In this case, it is used to separate the noise from the CH model by forming semantically meaningful groupings that contain points with similarities called clusters. Ideally, these clusters will correspond to a collection of points that make sense to humans, such as all the points that comprise the scaffolding in the three-dimensional church model. Therefore, the taxing task of point cloud cleaning can be significantly simplified if the aforementioned is achieved. The clustering techniques explored are partitional algorithms and a Gaussian mixture model. K-means is the baseline clustering algorithm in which the three aforementioned algorithms can be compared.

Research Objectives

  1. Does K-medians Gaussian mixture model and Fuzzy c-means produce clusters on point clouds of CH sites that are more semantically meaningful than those produced by k-means algorithm, using cluster validity metrics of Silhouette Coefficient and DB Index?
  2. What is the upper bound accuracy when using the resulting clusters to perform binary classification using' keep' and' discard' labels on a point cloud?
  3. Can clustering simplify and accelerate the mainly-manual point cloud cleaning task?

RESULTS

Clustering

K-means, K-medians, FCM and GMM can form semantically meaningful clusters on the raw dataset and the PointNet++ dataset. The algorithms fail on the CloudCompare dataset as the clusters are complex to decipher and do not correspond to specific sturctures of the CH site. No algorithm substantially outperforms any other algorithm in producing semantically meaningful clusters. Additionally, the prodcued clusters can simplify and accelerate the point cloud cleaning task as the clusters corresponding to noise can be easily discarded. Even though, there is sometimes overlap with points that do not belong to the CH site and points that do, it is still more simple and effective to remove some clusters and a few points instead of thousands of points.

Classification

The Gaussian mixture models maintains a strong performance in the classifcation task throughout all three datasets. It's performance is optimised when using the PointNet++ dataset. The k-means algorithm producing robust classification results however it's perfromance is based off 3000 clusters. Unfortunately, vast amounts of clusters can not help the point cloud cleaning task therefore we conclude that the GMM is superior.
